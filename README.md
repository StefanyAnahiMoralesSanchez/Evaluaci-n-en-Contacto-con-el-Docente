**Objetivo del proyecto:** 
El objetivo principaldel sistema es administrar libros digitales de manera organizada y eficiente, además de mantener un inventario consistente para lograr garantizar la trazabilidad de usuarios y transacciones.
Y como objetivos específicos permitir búsquedas más rápidas, gestionar usuarios y roles, controlar prestamos digitales y generar reportes simples.
**Funcionalidades principales:** 
Nuestro sistema de Gestión de Libros Electrónicos permite administrar de forma eficiente una biblioteca digital mediante el control de usuarios y contenidos. El sistema facilita el registro e inicio de sesión de los usuarios, la asignación de roles como administrador o lector y la gestión de sus datos personales. Además, permite agregar, editar y eliminar libros electrónicos, almacenarlos en formatos digitales como PDF y organizarlos en un catálogo digital clasificado por categorías, autores o fechas de publicación. Los usuarios pueden realizar búsquedas avanzadas para encontrar libros de manera rápida, visualizar o leer los libros en línea y, según sus permisos, descargarlos. El sistema también incorpora mecanismos de control de acceso para proteger el contenido digital, así como la generación de reportes y estadísticas que muestran información relevante como libros registrados, usuarios activos y títulos más consultados, contribuyendo a una gestión ordenada y moderna de los recursos digitales.
